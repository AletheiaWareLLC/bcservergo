BC Command Line Usage

Environment

  • $> ALIAS=alice ./bc

    Sets the alias for BC to load from the keystore.

  • $> ROOT_DIRECTORY=/tmp/foobar ./bc

    Sets the root directory for BC to use.

  • $> CACHE_DIRECTORY=/tmp/cache ./bc

    Sets the directory for BC to use for caching.

  • $> DEBUG=true ./bc

    Sets the debug flag for BC.

  • $> KEYS_DIRECTORY=~/keys ./bc

    Sets the directory for BC to use for storing keys.

  • $> CERTIFICATE_DIRECTORY=~/certificates ./bc

    Sets the directory for BC to use for storing certificates.

  • $> LOG_DIRECTORY=/tmp/logs ./bc

    Sets the directory for BC to use for logging.

  • $> PASSWORD=password1234 ./bc

    Sets the password for BC to unlock the private key from the keystore.

    Note: This is NOT advisable as your password will be stored in the command history, it is more secure to enter your password at the prompt.

  • $> PEERS=bc.aletheiaware.com,foobar.com ./bc

    Sets the peers for BC to communicate with.

Commands

  • $> ./bc

    Displays usage.

  • $> ./bc init

    Initializes environment, generates keys, and registers alias.

    $> ./bc init
    Creating keystore under /Users/username/bc
    Enter keystore password:
    Confirm keystore password:
    Generating RSA 4096 Public/Private Key Pair
    Successfully Created Key Pair
    ...
    ...
    Initialized as username 
  • $> ./bc node

    Display registered alias and public key.

  • $> ./bc alias [alias]

    Displays the public key registered with the given alias.

  • $> ./bc mine [channel] [threshold] [access...]

    Reads data from stdin and mines a new block in the given channel to the given given threshold. If the access list is empty the payload is unencrypted, else the payload is encrypted and the listed aliases are granted access.

  • $> ./bc head [channel]

    Displays the head of the given channel.

  • $> ./bc block [channel] [hash]

    Displays the block in the given channel with the given hash.

  • $> ./bc record [channel] [hash]

    Displays the record in the given channel with the given hash.

  • $> ./bc registration [merchant]

    Displays the registration information between this alias and the given merchant.

  • $> ./bc subscription [merchant]

    Displays the subscription information between this alias and the given merchant.

  • $> ./bc pull [channel]

    Pulls the given channel from peers.

  • $> ./bc push [channel]

    Pushes the given channel to peers.

  • $> ./bc import-keys [alias] [access-code]

    Import Alias Keys from another device.

  • $> ./bc export-keys [alias]

    Export Alias Keys to another device.

  • $> ./bc keystore

    Display location of keystore.

  • $> ./bc peers

    Display list of peers.

  • $> ./bc add-peer [host]

    Add given host to list of peers.

  • $> ./bc cache

    Display location of cache.

  • $> ./bc purge

    Purges all data from cache.

  • $> ./bc random

    Generate a random number.
