Sample Usage

Environment 

  • $> ALIAS=alice ./bc

    Sets the alias for BC to load from the keystore.

  • $> CACHE=/tmp/cache ./bc

    Sets the directory for BC to use for caching.

  • $> KEYSTORE=~/keys ./bc

    Sets the directory for BC to use for storing keys.

  • $> PASSWORD=password1234 ./bc

    Sets the password for BC to unlock the private key from the keystore.

    Note: This is NOT advisable as your password will be stored in the command history, it is more secure to enter your password at the prompt.

  • $> PEERS=bc.aletheiaware.com ./bc

    Sets the peers for BC to communicate with.

Commands 

  • $> ./bc alias [alias]

    Displays the public key registered with the given alias

  • $> ./bc block [channel] [hash]

    Displays the block in the given channel with the given hash

  • $> ./bc head [channel]

    Displays the head of the given channel

  • $> ./bc init

    Initializes environment, generates keys, and registers alias.

  • $> ./bc node

    Display registered alias and public key

  • $> ./bc record [channel] [hash]

    Displays the record in the given channel with the given hash

  • $> ./bc sync [channel]

    Synchronizes cache for the given channel

  • $> ./bc import-keys [alias] [access-code]

    Import Alias Keys from another device

  • $> ./bc export-keys [alias]

    Export Alias Keys to another device

  • $> ./bc keystore

    Display location of keystore

  • $> ./bc peers

    Display list of peers

  • $> ./bc add-peer [host]

    Add given host to list of peers

  • $> ./bc cache

    Display location of cache

  • $> ./bc purge

    Purges all data from cache

  • $> ./bc random

    Generate a random number
